# Smart Money 13F

A Streamlit dashboard that scrapes SEC 13F filings to track long-only institutional portfolios, top holdings, portfolio weights, and quarter-over-quarter changes.

## Features
- Pull 13F filings directly from SEC
- Parse holdings into structured tables
- Show top 10 holdings by portfolio weight
- Highlight quarterly changes in positions
- Sector exposure analysis
- Streamlit dashboard for portfolio research
